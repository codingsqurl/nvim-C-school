# 000_WEDEV_MOC

> Web development - HTML, CSS, JavaScript, and full-stack engineering.

## Overview

WEDEV covers modern web development from markup to full-stack applications:

- [[001_HTML_Fundamentals]] - Semantic HTML, accessibility, forms, SEO
- [[002_CSS_Layout_Mastery]] - Flexbox, Grid, responsive design, animations
- [[003_JavaScript_Core]] - JS fundamentals, ES6+, closures, prototypes, promises
- [[004_DOM_Events]] - DOM manipulation, event delegation, browser APIs
- [[005_Frontend_Frameworks]] - React/Vue/Svelte concepts, components, state management
- [[006_Backend_APIs]] - Node.js, Express, REST, GraphQL, middleware
- [[007_Databases_Storage]] - SQL, NoSQL, ORMs, caching strategies
- [[008_Full_Stack_Projects]] - Auth flows, testing, CI/CD, deployment, monitoring
